Thêm ngôn ngữ mới
**** File tiếng việt đã có sẵn trong code và đã được cài đặt sẵn!
Nếu không có ngôn ngữ tiếng Việt, bạn chỉ cần thêm ngôn ngữ trên trang web bằng cách: Vào Bảng quản trị ( Admin Panel ) [truy cập] Ngôn ngữ ( Language ) [truy cập] Thêm ngôn ngữ ( Add New Language ).
  >  > Cách thêm ngôn ngữ tiếng việt
     B1. Sau khi vào " Add New Language ". Tại ô "Default" nếu bạn muốn đó là ngôn ngữ mặc đinh thì bạn bật lên hoặc không thì khỏi bật nhé.
     B2. Tại ô "Enabled" nếu bạn đã chỉnh hoàn chỉnh ngôn ngữ bạn bật để người khác có thể dùng. Tắt để không một ai có thể dùng ngôn ngữ đó.
     B3. Tại ô "Code" với ngôn ngữ tiếng Việt bạn ghi là vi_vn . Và các ngôn ngữ khác bạn tra trên Google sẽ ra nhé.
     B4. Tại ô "Native Name" là ngôn ngữ bạn đang thêm, ví dụ "Tiếng Việt".
     B5. Tại ô "Flag" bạn thêm cờ (quốc kỳ) của ngôn ngữ/quốc gia đó. Với cờ Việt Nam có trong "File Tiếng Việt" > "cờ_vietnam". Bạn có thể lấy ra để sử dụng.
  >  > Các ngôn ngữ khác cũng tương tự nhé ^^
Tải phần mềm Poedit trước khi thực hiện tiếp.
Cài đặt ngôn ngữ mới bằng cách sao chép thư mục "Translation" > > "src" ra màn hình máy tính của bạn và thực hiện chỉnh sửa ngôn ngữ tại file "messages.po" và không can thiệp đến file "messages.mo" nhé.
