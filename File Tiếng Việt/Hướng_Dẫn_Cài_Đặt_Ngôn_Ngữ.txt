Ngôn ngữ tiếng Việt, mặc định được cài trong mã nguồn, nếu không xuất hiện, hãy thực hiện các bước dưới đây!
=====
(Nếu chưa có 2 file ở B4 trong Hosting)
B1. Truy cập trang quản lý máy chủ / hosting.
B2. Vào trang lưu trữ mã nguồn và làm tiếp.
B3. Sau khi vào trang lưu trữ source code, bạn  truy cập content/languages/vi_vn/LC_Messages
B4. Upload 2 file "messages.po" và "messages.mo" trong thư mục vi_vn/LC_Messages.
Vậy là hoàn tất cách cài.
=====
(Nếu đã có 2 file ở B4 trên trong hosting)
B1. Đăng nhập mạng xã hội của bạn, sau đó tTruy cập đường dẫn: tên miền của bạn/admincp/languages/add (Ví dụ: tcsn.tcteamcorp.com/admincp/languages)
B2. Chỉnh những tùy chọn theo mẫu dưới đây:
- Default (Mặc định): Bật
- Enabled (Kích hoạt): Bật
- Code (Mã): vi_vn
- Native Name (Tên gốc): Tiếng Việt
- Flag (Quốc kỳ): Upload file vi_vn.png lên
- Direction (Điều hướng): LTR
- Order (Số thứ tự): Số mấy cũng được
